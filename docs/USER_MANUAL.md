## Getting Started

Welcome to ComSpace! This guide will help you navigate and use all features of our e-commerce platform.

### System Requirements
- **Web**: Modern browser (Chrome, Firefox, Safari, Edge)
- **Mobile**: iOS 13+ or Android 8+
- **Internet**: Stable connection recommended

### First Time Access
1. Visit the website or download the mobile app
2. Allow location access for automatic currency detection (optional)
3. Browse products as a guest or create an account for full features

---

## Creating an Account

### Registration Methods

#### Email Registration
1. Click **"Sign Up"** or **"Register"**
2. Fill in the form:
   - First Name
   - Last Name
   - Email Address
   - Password (minimum 8 characters)
   - Phone Number (optional)
3. Click **"Create Account"**
4. Check your email for verification link
5. Click the verification link to activate your account

#### Social Login
1. Click **"Sign in with Google"** or **"Sign in with Apple"**
2. Authorize the application
3. Complete any additional profile information
4. Start shopping!

### Email Verification
- Check your inbox for verification email
- Click the link within 24 hours
- If not received, check spam folder
- Request new link from login page

## Browsing Products

### Product Catalog
- **Home Page**: Featured products and categories
- **Products Page**: Full catalog with filters
- **Categories**: Browse by product type
- **Search**: Use search bar for specific items

### Filtering Products
1. Navigate to Products page
2. Use sidebar filters:
   - **Price Range**: Set min/max prices
   - **Category**: Select product categories
   - **Rating**: Filter by customer ratings
   - **Availability**: In stock only
3. Click **"Apply Filters"**

### Product Details
Click any product to view:
- High-resolution images (zoom available)
- Detailed description
- Price in your local currency
- Stock availability
- Customer reviews and ratings
- Shipping information
- Size/color variants (if applicable)

### Search Tips
- Use specific keywords
- Try partial names
- Use filters to narrow results
- Check spelling

## Shopping Cart

### Adding Items
1. Select product
2. Choose quantity, size, color (if applicable)
3. Click **"Add to Cart"**
4. View cart icon for item count

### Managing Cart
- **View Cart**: Click cart icon (top right)
- **Update Quantity**: Use +/- buttons
- **Remove Items**: Click trash icon
- **Save for Later**: Move items to wishlist
- **Clear Cart**: Remove all items

### Cart Summary
- Subtotal: Sum of all items
- Tax: Calculated based on location
- Shipping: Based on delivery method
- **Total**: Final amount in your currency

## Checkout Process

### Step 1: Shipping/Pickup Selection

#### Delivery Option
1. Select **"Delivery"**
2. Enter or select shipping address:
   - Street Address
   - City, State, ZIP
   - Country
3. Address validation occurs automatically
4. View estimated delivery date
5. See shipping cost

#### Pickup Option
1. Select **"Store Pickup"**
2. System detects nearby stores
3. Choose pickup location
4. Select pickup date/time
5. View store hours and contact

### Step 2: Review Order
- Verify items, quantities, prices
- Confirm delivery/pickup details
- Check total amount
- Apply coupon codes (if any)

### Step 3: Payment
1. Select payment method
2. Enter payment details (encrypted)
3. Review final amount
4. Click **"Place Order"**

### Order Confirmation
- Confirmation number displayed
- Email receipt sent immediately
- Order appears in **"My Orders"**

## Order Tracking

### Viewing Orders
1. Login to your account
2. Click **"My Orders"** or **"Profile"** → **"Orders"**
3. View all orders with status

### Order Status
- **Pending**: Payment processing
- **Confirmed**: Order accepted
- **Processing**: Being prepared
- **Shipped**: On the way (delivery)
- **Out for Delivery**: Arriving today
- **Delivered**: Completed
- **Ready for Pickup**: Available at store
- **Picked Up**: Completed

### Tracking Shipments
1. Open order details
2. Click **"Track Shipment"**
3. View real-time location
4. See estimated delivery time
5. Receive notifications:
   - Order confirmed
   - Shipped
   - Out for delivery
   - Delivered

### Canceling Orders
1. Go to order details
2. Click **"Cancel Order"** (if available)
3. Confirm cancellation
4. Refund processed within 5-7 business days

**Note**: Orders cannot be canceled once shipped or ready for pickup.

## Profile Management

### Updating Profile
1. Click **"Profile"** or user icon
2. Edit information:
   - Name
   - Email
   - Phone
   - Avatar/Photo
3. Click **"Save Changes"**

### Managing Addresses
1. Go to **"Profile"** → **"Addresses"**
2. Add new address:
   - Click **"Add Address"**
   - Fill in details
   - Set as default (optional)
3. Edit existing addresses
4. Delete unused addresses

### Changing Password
1. Go to **"Profile"** → **"Security"**
2. Enter current password
3. Enter new password (2x)
4. Click **"Update Password"**

### Notification Preferences
1. Go to **"Profile"** → **"Notifications"**
2. Toggle preferences:
   - Email notifications
   - Push notifications (mobile)
   - SMS alerts
   - Marketing emails
3. Save changes

### Order History
- View all past orders
- Download invoices
- Reorder previous items
- Leave product reviews

## Troubleshooting

### Cannot Login
- **Forgot Password**: Click "Forgot Password" link
- **Account Locked**: Contact support after multiple failed attempts
- **Email Not Verified**: Check inbox and verify email

### Payment Declined
- Verify card details
- Check sufficient funds
- Contact your bank (may block online purchases)
- Try different payment method
- Check billing address matches card

### Items Out of Stock
- Add to wishlist for notifications
- Check back periodically
- Contact support for restock info

### Wrong Currency Displayed
- Manually select correct currency
- Clear browser cache
- Allow location access
- Contact support if persists

### Order Not Received
1. Check order status in account
2. Verify delivery address
3. Check with delivery service
4. Contact support with order number

### Returns and Refunds
1. Go to order details
2. Click **"Request Return"**
3. Select reason
4. Upload photos if damaged
5. Print return label
6. Ship within 30 days
7. Refund processed upon receipt
